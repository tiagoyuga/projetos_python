Teste 128Bits - Descrição do teste.

Descrição:
Você deverá criar um sistema de busca que se aproxime o máximo das especificações e instruções abaixo:
  • 1 - Primeiramente, deverá ser criado um modelo. Um modelo base que terá dois atributos, 1 inteiro e um de texto.

  • 2 - Deverá ser criado um CRUD utilizando as Class-Based Views do Django. (OK) Sendo assim será possível listar os items (paginada por 100 items) do modelo base,(ok) criar novos items, deletar items assim como atualizá-los.(OK)

  • 3 - Deverá haver um campo de busca com "autocomplete" no topo. Quando o usuário digitar 3 letras deverá ser trazido no "autocomplete" os items com esses caracteres dispostos consecutivamente no campo de texto. Exemplo de biblioteca:(django_selectable, )

  • 4 - Os valores inteiros deverão ser gerados randomicamente. Já os valores de texto deverão ser obtidos através da api (http://api.randomuser.me/).

  • 5 - O código deverá ser disposto no Github.(link GitHub: - https://github.com/tiagoyuga/projetos_python.git)

  • 6 - BONUS: Provisionar um servidor (Amazon/Heroku/DigitalOcean) e realizar o deploy*.
